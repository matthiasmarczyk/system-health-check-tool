# System Health Check Tool

A simple Python command-line tool that displays basic system information.

This project was created as part of my learning journey into IT support and Python programming.

## Features

This tool can:

- display operating system information
- show computer name, processor, and Python version
- test internet connectivity
- display disk usage information
- provide a basic RAM-related check note

## Technologies

- Python
- platform
- subprocess
- shutil

## How to run

Run the script from the terminal:

```bash
py src/system_check.py
```
