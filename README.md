# System Health Check Tool

A small Python project that displays basic system information.

## Features

This tool shows:

- operating system
- computer name
- processor
- Python version

## Technologies

- Python
- platform

## How to run

Run the script from the terminal:

```bash
py src/system_check.py
```

## Purpose

I built this project to improve my Python skills and learn more about system-related information in IT.
